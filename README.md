# Matchora Backend

Anonymous connection platform API built with Node.js, Express, and MongoDB.

## Features

- JWT Authentication (Signup/Login)
- User Profiles with Anonymous Names
- Smart Matching Algorithm (100-point compatibility score)
- Real-time Chat via Socket.io
- MongoDB Atlas Integration
- Content Moderation & Safety Features

## Tech Stack

- **Runtime:** Node.js 18+
- **Framework:** Express.js
- **Database:** MongoDB Atlas
- **Real-time:** Socket.io
- **Auth:** JWT + bcrypt
